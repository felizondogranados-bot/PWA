    🎓 Explicación Técnica (PWA Info) ▼

    🛠️ Service Worker

    Es un script que el navegador ejecuta en segundo plano. Permite interceptar peticiones de red, manejar la caché y posibilitar la carga instantánea y el soporte sin conexión.

    Ver archivo: service-worker.js

    💾 Estrategia de Caché

    Usamos Cache-First. Al abrir la app, el Service Worker instala y guarda en la caché del navegador los archivos HTML, CSS, JS e iconos. Si no hay conexión, se sirven directo desde la caché.

    📱 Web App Manifest

    Archivo JSON que especifica el nombre de la app, iconos, pantalla de inicio, modo standalone y color de fondo. Permite que el sistema operativo la reconozca como una aplicación nativa instalable.

    Ver archivo: manifest.json

    ⚡ Persistencia Local

    Las tareas se guardan en localStorage del navegador. Aunque se cierre la pestaña o el dispositivo se reinicie, los datos persisten localmente en formato JSON.
